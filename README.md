## 🚀 Overview

BDH Linux is a custom developer workspace designed to eliminate the hassle of manual configuration. Install once and start coding immediately!

> ⚠️ Currently in **Demo/Alpha** phase — setup completes in 4 to 10 commands.

---

## ✨ Key Features

| Feature | Description |
|---|---|
| 🎨 Cyberpunk Theme | Sleek dark terminal aesthetic |
| 🐍 Pre-configured Stack | Python 3.12+, FastAPI, PostgreSQL ready |
| 🤝 BDH Custom ZSH Prompt | Tailored shell for better productivity |
| ⌨️ Power Aliases | `serve`, `venv`, `gp` shortcuts |
| 🚀 FastAPI Ready | Instant project scaffolding |
| ⚡ Bloat-free | Maximum performance |

---

## 🛠️ The Stack

- **Base OS:** Manjaro (Arch-based)
- **Shell:** ZSH + Oh My Zsh
- **Language:** Python 3.12+
- **Framework:** FastAPI
- **Database:** PostgreSQL
